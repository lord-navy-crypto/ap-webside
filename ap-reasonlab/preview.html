如何启动完整网站

  1. 安装 Node.js 20+：nodejs.org
  2. 打开终端，进入项目目录：cd ap-reasonlab
  3. 安装依赖：npm install
  4. 启动开发服务器：npm run dev
  5. 浏览器打开：http://localhost:3000
